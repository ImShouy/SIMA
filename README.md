Andeva, es una startup peruana agrotech creada por estudiantes de ingeniería de software de la Universidad Peruana de Ciencias Aplicadas (UPC)
con una misión clara y concisa: que ningún agricultor de las regiones altoandinas vuelva a decidir “a ciegas”. 
Nuestro producto principal es SIMA (Sistema Inteligente de Monitoreo Agrícola), una plataforma que busca cerrar la brecha de acceso a la información agraria a los agricultores de regiones altoandinas.
SIMA consiste en el uso de diferentes tipos de dispositivos IoT en las parcelas de los agricultores, conexión a través de LoRaWAN, procesamiento y almacenamiento de datos en la nube.
Además, mediante técnicas de Machine Learning con los datos locales y de todas las parcelas registradas, genera reportes personalizados diarios, semanales, mensuales o anuales, según la preferencia del usuario.
